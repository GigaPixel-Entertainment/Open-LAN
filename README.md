# GPE-Open-LAN
______________
Open-LAN is an open-source, LAN messaging system by GigaPixel Entertainment.

We published the code here on GitHub so everyone can set up LAN messaging systems, at their organization or home.

# Usage
_______
1. Open-LAN is somewhat similar to Discord both in features and UI, being that the server is self-hosted by your organization.

Before using Open-LAN, ensure you are connected to your home's/organization's internet connection and have an Open-LAN account as both are required to use Open-LAN. No email, phone number, or any form of identification other than your name is needed to create an account. 

To use Open-LAN, type in your network IP address, and log in to your account to begin messaging.

2. Simply type https://www.openlan.gigapixel.cc for the public web version of Open-LAN
