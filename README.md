# GPE-Open-LAN
______________
Open-LAN is an open-source, LAN messaging system by GigaPixel Entertainment.

We published the code here on GitHub so everyone can set up LAN messaging systems, at their organization or home.
# Usage
_______
Open-LAN is somewhat similar to Discord both in features and UI, with the only difference being that the server is self-hosted by your organization.

Before using Open-LAN, ensure you are connected to your home's/organization's internet connection, as it is needed to use Open-Lan. To use Open-Lan, type in your network IP address, and log in to your account to begin messaging.

Most actions will be run on the server side and no downloads are required for client.
