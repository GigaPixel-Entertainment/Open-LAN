Open-LAN Terms of Service

  1. Please report any bugs and/or inappropriate behaviors of the Open-LAN program, as we’d like feedback on what needs to be fixed with the Open-LAN project. Please report any bugs on GitHub.
  2. If there are any bugs with Open-LAN, please do not exploit them, and instead take the proper actions as stated in rule number 1.
  3. Please be respectful when using Open-LAN.
  4. Do not collect or sell data of any users using the public, web version of Open-LAN.
  5. Do not use any information or data from the Open-LAN public version to train Artificial Intelligence models.
  6. Do not make any form of plugin or paid feature on the public Open-LAN platform, as we aim to keep the majority/nearly all of things in Open-LAN free. If GigaPixel Entertainment does include future paid features to public Open-LAN, it is probably so we can afford server costs, if any arise.
  7. Do not make any bots on public Open-LAN with the intention of spam or malware.
